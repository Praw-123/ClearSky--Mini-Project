## 🛠 เทคโนโลยีที่ใช้

| เทคโนโลยี | รายละเอียด |
|---|---|
| React Native | Framework หลักสำหรับพัฒนา Mobile App |
| Expo | Platform สำหรับ build และ run แอป |
| JavaScript (ES6+) | ภาษาที่ใช้เขียนโค้ด |
| OpenWeatherMap API | แหล่งข้อมูลสภาพอากาศ |
| Figma | ออกแบบ UI ก่อน implement |
| Git + GitHub | Version control |

## 🚀 วิธีรันโปรเจค

### สิ่งที่ต้องติดตั้งก่อน
- [Node.js](https://nodejs.org)
- [Expo Go](https://expo.dev/client) บนมือถือ

### ขั้นตอน

```bash
# 1. Clone โปรเจค
git clone https://github.com/ชื่อ-github-เรา/ClearSky2.git

# 2. เข้าโฟลเดอร์
cd ClearSky2

# 3. ติดตั้ง dependencies
npm install

# 4. รันโปรเจค
npx expo start
```

5. สแกน QR Code ด้วยแอป Expo Go บนมือถือ

## 🔑 ตั้งค่า API Key

1. สมัครบัญชีฟรีที่ [openweathermap.org](https://openweathermap.org)
2. คัดลอก API Key จากหน้า My API Keys
3. เปิดไฟล์ `App.js` แล้วแทนที่บรรทัดนี้

```javascript
const API_KEY = 'ใส่ API Key ของคุณตรงนี้';
```

## 📚 สิ่งที่ได้เรียนรู้จากโปรเจคนี้

- การพัฒนา Mobile Application ด้วย React Native และ Expo
